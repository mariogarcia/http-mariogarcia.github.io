== Deploy my first pod

=== Deploy pod

[source, shell]
----
kubectl create -f nodejs-pod.yml
----

=== Describe pod

[source, shell]
----
kubectl describe pods/nodejs-pod.yml
----

=== Destroy pod

[source, shell]
----
kubectl delete pods nodejs-pod.yml
----
